## 使用检查

第一次打开文档站时，建议先确认：

1. 从“快速开始”能启动并构建文档站。
2. 从“学习路线”能按角色找到第一批要读的文档。
3. 从“按模块学习”能看到账号、安全、学习、OJ、AI、内容和后台的连续阅读顺序。
4. 从“模块学习任务模板”能看到每一站的作业单、过关标准和回写模板。
5. 从“上手任务”能选择一个角色任务包，并看到输出模板。
6. 从“查看模块”能进入全量模块总览。
7. 本地搜索能找到核心模块，例如 OJ、AI Runtime、文件存储。
8. 模块页能继续跳到 API、数据表、源码地图和操作手册。

# 开发者工具

开发者工具主要是用户端前端工具集合，面向日常开发辅助场景。

## 功能入口

| 页面 | 说明 |
| --- | --- |
| `/dev-tools` | 工具首页 |
| `/dev-tools/json` | JSON 格式化、压缩、校验等 |
| `/dev-tools/text-diff` | 文本对比 |
| `/dev-tools/translation` | 翻译工具 |

## 当前定位

开发者工具以轻量、即时、低依赖为主。能在前端完成的能力优先前端完成，需要保存历史、调用 AI 或访问外部服务时再接后端。

## 后续扩展方向

- Base64、URL 编解码。
- 时间戳转换。
- 正则测试。
- JWT 解析。
- Markdown 预览。
- SQL 格式化。
- Cron 表达式解释。
- 代码片段收藏。

## 开发注意

- 工具页要避免把用户输入自动上传到服务端。
- 如果接入 AI 翻译或解释，需要明确提示数据会被发送到模型服务。
- 大文本处理要考虑浏览器性能。

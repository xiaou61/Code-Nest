# 工具、摸鱼与版本

这一组能力偏向轻工具、运营内容和产品版本展示。

## 功能地图

| 功能 | 用户端入口 | 管理端入口 | 后端模块 |
| --- | --- | --- | --- |
| 开发者工具 | `/dev-tools`、`/dev-tools/json`、`/dev-tools/text-diff`、`/dev-tools/translation` | 后续扩展 | 前端工具为主 |
| 摸鱼工具 | `/moyu-tools` | `/moyu/*` | `xiaou-moyu` |
| 程序员日历 | `/moyu-tools/calendar` | `/moyu/calendar-events` | `xiaou-moyu` |
| 热榜和每日内容 | `/moyu-tools/hot-topics`、`/moyu-tools/daily-content` | `/moyu/daily-content` | `xiaou-moyu` |
| 薪资计算器 | `/moyu-tools/salary-calculator` | 后续扩展 | `xiaou-moyu` |
| Bug 商店 | `/moyu-tools/bug-store` | `/moyu/bug-store` | `xiaou-moyu` |
| 版本历史 | `/version-history` | `/system/version` | `xiaou-version` |

# 计划与学习小组

计划模块解决个人执行，小组模块解决协作学习。两者共同支撑长期成长任务。

## 功能入口

| 功能 | 用户端入口 | 后端模块 |
| --- | --- | --- |
| 个人计划 | `/plan` | `xiaou-plan` |
| 成长自动驾驶 | `/growth-autopilot`，当前重定向到 `/learning-cockpit` | `xiaou-plan`、`xiaou-ai` |
| 学习小组 | `/team`、`/team/create`、`/team/my`、`/team/:id`、`/team/:id/edit` | `xiaou-team` |

## 个人计划接口域

| 接口域 | 能力 |
| --- | --- |
| `/user/plan` | 创建、更新、删除、详情、列表、暂停、恢复、今日任务、打卡、统计 |
| `/user/plan/autopilot` | 自动驾驶仪表盘、生成计划、重新规划、完成任务、延期任务 |

## 学习小组接口域

`/user/team` 覆盖小组生命周期和协作能力：

- 创建、更新、删除、详情、列表、推荐。
- 我的参与和我创建的小组。
- 邀请码、刷新邀请码、按邀请码加入。
- 加入、退出、申请、审批、拒绝、取消。
- 成员移除、角色调整、转让、禁言。
- 小组任务创建、更新、删除、状态、今日任务。
- 打卡、补卡、删除、详情、列表、我的打卡、日历。
- 打卡点赞、连续天数、总量、排行榜。
- 讨论创建、更新、删除、详情、列表、置顶、精华、点赞。
- 小组统计、周统计、月统计、我的统计。

## 开发注意

- 小组权限要区分创建者、管理员、成员和非成员。
- 邀请码加入要考虑过期、刷新和枚举风险。
- 打卡、补卡、连续天数和排行榜需要明确时间口径。
- 自动生成计划必须保留人工调整入口。

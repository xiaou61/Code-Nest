## 🚀 安装步骤

### 1. 环境准备
<!-- 描述环境准备步骤 -->

```bash
# 安装Java
yum install -y java-11-openjdk

# 安装MySQL
yum install -y mysql-server

# 安装Redis
yum install -y redis
```

### 2. 应用部署
<!-- 描述应用部署步骤 -->

```bash
# 下载应用
wget https://example.com/app.jar

# 启动应用
java -jar app.jar --spring.profiles.active=prod
```

### 3. 配置说明
<!-- 描述配置说明 -->

```yaml
# application-prod.yml
server:
  port: 8080
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/db
    username: root
    password: password
```

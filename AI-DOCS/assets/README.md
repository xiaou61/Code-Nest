# assets (公共资源)

## 目录说明

本目录存放文档相关的公共资源，包括图片、模板等。

## 目录结构

```
assets/
├── images/                # 图片资源
│   ├── 模块名称/          # 按模块组织
│   ├── 架构图/            # 架构相关图片
│   └── 流程图/            # 流程相关图片
├── templates/             # 文档模板
│   ├── PRD-template.md
│   ├── Technical-template.md
│   ├── Testing-template.md
│   └── ...
└── README.md              # 本文件
```

## 资源规范

### 图片规范
- **格式**: PNG, JPG, SVG
- **命名**: `{模块}-{功能}-{描述}.png`
- **大小**: 单张图片不超过2MB
- **分辨率**: 适合屏幕显示

### 模板规范
- **格式**: Markdown
- **命名**: `{文档类型}-template.md`
- **内容**: 包含完整的文档结构

## 图片管理

### 图片分类
- 架构图：系统架构、模块架构
- 流程图：业务流程、技术流程
- 截图：界面截图、操作截图
- 图表：数据图表、统计图表

### 图片优化
- 压缩图片大小
- 使用合适的格式
- 添加Alt文本
- 保持清晰度

## 模板管理

### 模板类型
- PRD模板
- 技术文档模板
- 测试文档模板
- 部署文档模板
- 用户文档模板
- 开发文档模板

### 模板使用
- 复制模板文件

- 保持模板结构
- 更新模板版本

## 资源引用

### 相对路径
```markdown
![图片描述](images/模块名称/图片.png)
```

### 绝对路径
```markdown
![图片描述](/AI-DOCS/assets/images/模块名称/图片.png)
```

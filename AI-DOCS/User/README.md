# User (用户文档)

## 目录说明

本目录存放所有用户相关文档，包括用户手册、操作指南、FAQ等。

## 目录结构

```
User/
├── 用户手册/              # 用户使用手册
│   ├── 功能说明.md
│   ├── 使用教程.md
│   └── 常见问题.md
├── 操作指南/              # 操作指南
│   ├── 功能操作.md
│   ├── 流程说明.md
│   └── 截图指南.md
├── FAQ/                   # 常见问题
│   ├── 功能FAQ.md
│   ├── 技术FAQ.md
│   └── 账号FAQ.md
└── README.md              # 本文件
```

## 文档规范

### 命名规范
- **用户手册**: `{功能名称}使用手册.md` (如: `积分抽奖使用手册.md`)
- **操作指南**: `{功能名称}操作指南.md` (如: `积分抽奖操作指南.md`)
- **FAQ**: `{主题}FAQ.md` (如: `积分抽奖FAQ.md`)

### 文档模板
参考 `assets/templates/User-template.md`

# Development (开发文档)

## 目录说明

本目录存放所有开发相关文档，包括开发规范、环境搭建、贡献指南等。

## 目录结构

```
Development/
├── 开发规范/              # 编码规范、Git规范等
│   ├── 编码规范.md
│   ├── Git规范.md
│   └── 代码审查.md
├── 环境搭建/              # 开发环境搭建
│   ├── Java环境.md
│   ├── 前端环境.md
│   └── 数据库环境.md
├── 贡献指南/              # 如何贡献代码
│   ├── 贡献流程.md
│   ├── 代码规范.md
│   └── 提交规范.md
└── README.md              # 本文件
```

## 文档规范

### 命名规范
- **开发规范**: `{主题}规范.md` (如: `Java编码规范.md`)
- **环境搭建**: `{环境}搭建指南.md` (如: `Java开发环境搭建指南.md`)
- **贡献指南**: `{主题}指南.md` (如: `代码贡献指南.md`)

# v2.0.1 功能升级实施计划

## 背景

v2.0.1 基于 `master` 新开分支进行升级，版本号升级已经完成。本计划覆盖本次增量开发的四个功能点，优先复用现有模块与接口，避免引入高风险表结构迁移。

## 功能范围

1. **AI Runtime 质量治理增强**
   - 在 `xiaou-ai` 提供轻量治理摘要能力，输出工作流目录、fallback 覆盖、治理建议等运行态信息。
   - 在管理端系统页面增加 AI 治理视图，方便管理员快速检查 AI 能力接入状态。

2. **求职闭环二期**
   - 扩展求职闭环阶段，补齐面试复盘后的投递/Offer 跟踪动作。
   - 在用户端求职闭环页展示新的阶段状态与行动入口。

3. **OJ 赛事增强**
   - 在赛事榜单增加赛后评分预估字段：表现分、评分变化、赛后预估评分。
   - 前端赛事详情榜单展示评分变化，作为不新增评分表前的轻量赛事反馈。

4. **小体验补齐**
   - 管理端文件管理支持文本类文件直接预览。
   - 管理端积分明细支持按当前筛选条件导出 CSV。

## 验证计划

- 后端：`mvn -pl xiaou-application -am clean package -DskipTests`
- 管理端：`npm --prefix vue3-admin-front run build`
- 用户端：`npm --prefix vue3-user-front run build`

# Archive (归档文档)

## 目录说明

本目录存放已归档的历史文档和废弃功能文档。

## 目录结构

```
Archive/
├── 历史版本/              # 历史版本文档
│   ├── PRD/               # 历史PRD
│   ├── 技术文档/          # 历史技术文档
│   └── 测试文档/          # 历史测试文档
├── 废弃功能/              # 废弃功能文档
│   ├── 功能名称/          # 按功能组织
│   └── README.md
└── README.md              # 本文件
```

## 文档规范

### 命名规范
- **历史文档**: 保持原始文件名
- **废弃文档**: `{功能名称}废弃说明.md`

### 归档规则
- 版本更新后，旧版本文档移至历史版本
- 功能废弃后，相关文档移至废弃功能
- 归档文档保持只读状态

## 归档流程

### 版本归档
1. 创建新版本文档
2. 将旧版本文档移至历史版本
3. 更新文档索引

### 功能归档
1. 创建功能废弃说明
2. 将相关文档移至废弃功能
3. 更新文档索引

## 文档保留策略

### 保留期限
- 历史版本文档：保留最近3个版本
- 废弃功能文档：保留1年
- 临时文档：保留3个月

### 清理规则
- 定期清理过期文档
- 重要文档永久保留
- 敏感文档安全删除

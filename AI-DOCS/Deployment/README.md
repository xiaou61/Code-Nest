# Deployment (部署文档)

## 目录说明

本目录存放所有部署和运维相关文档，包括部署指南、运维手册、监控告警等。

## 目录结构

```
Deployment/
├── 部署指南/              # 部署相关
│   ├── Docker部署.md
│   ├── K8s部署.md
│   └── 本地部署.md
├── 运维手册/              # 运维相关
│   ├── 日常运维.md
│   ├── 故障处理.md
│   └── 备份恢复.md
├── 监控告警/              # 监控相关
│   ├── Prometheus监控.md
│   ├── Grafana仪表盘.md
│   └── 告警规则.md
└── README.md              # 本文件
```

## 文档规范

### 命名规范
- **部署文档**: `{部署方式}部署指南.md` (如: `Docker部署指南.md`)
- **运维文档**: `{主题}手册.md` (如: `日常运维手册.md`)
- **监控文档**: `{监控工具}配置.md` (如: `Prometheus配置.md`)

### 文档模板
参考 `assets/templates/Deployment-template.md`

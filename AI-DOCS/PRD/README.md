# PRD (产品需求文档)

## 目录说明

本目录存放所有产品需求文档（Product Requirements Document），按模块组织。

## 目录结构

```
PRD/
├── 模块名称/              # 按模块组织PRD
│   ├── README.md          # 模块PRD索引
│   ├── v1.0.0.md          # 版本化PRD
│   └── assets/            # PRD相关图片资源
└── README.md              # 本文件
```

## 文档规范

### 命名规范
- **文件名**: `{模块名称}PRD-v{版本号}.md`
- **示例**: `积分抽奖模块PRD-v1.0.0.md`

### 版本管理
- 使用语义化版本号 (如: v1.0.0, v1.1.0, v2.0.0)
- 重大更新创建新版本文件
- 旧版本移至 `Archive/历史版本/` 目录

### 文档模板
参考 `assets/templates/PRD-template.md`

# 测试用例

## 目录说明

本目录存放各模块的测试用例文档。

## 测试用例模板

### 用例格式

```
用例ID: TC-模块-功能-序号
用例名称: [用例名称]
优先级: [P0/P1/P2/P3]
前置条件: [前置条件]
测试步骤:
  1. [步骤1]
  2. [步骤2]
  3. [步骤3]
预期结果: [预期结果]
测试数据: [测试数据]
```

### 优先级说明

| 优先级 | 说明 | 测试时机 |
|--------|------|---------|
| P0 | 核心功能 | 每次发布必须测试 |
| P1 | 重要功能 | 每周测试 |
| P2 | 一般功能 | 每月测试 |
| P3 | 边缘功能 | 按需测试 |

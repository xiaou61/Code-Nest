## 测试报告模板

### 报告格式

```
测试报告

1. 测试概述
   - 项目名称: [项目名称]
   - 版本号: [版本号]
   - 测试周期: [开始日期] 至 [结束日期]
   - 测试人员: [人员列表]

2. 测试执行
   - 用例总数: [数量]
   - 通过数: [数量]
   - 失败数: [数量]
   - 阻塞数: [数量]
   - 通过率: [百分比]

3. 缺陷统计
   - 缺陷总数: [数量]
   - 致命: [数量]
   - 严重: [数量]
   - 一般: [数量]
   - 轻微: [数量]

4. 遗留问题
   - [问题1]
   - [问题2]

5. 测试结论
   - [结论]
```
